# Customer Cases Assets

这个目录专门给官网 `客户案例` 页面放公开素材。

当前规则：

- 官网只放公开版素材
- 原始截图先做重命名
- 有地址栏、IP、住户信息、设备编号、坐标等内容时，先脱敏再上站

## 目录说明

- `screenshots/`：放截图
- `videos/`：放视频

## 当前命名规则

- 截图：`case-01-*.png`、`case-03-*.jpg`
- 视频：`case-01-demo.mp4`

## 当前案例分组

- `case-01-*`：无线网络质量评估
- `case-02-*`：运维监控与服务器管理
- `case-03-*`：数字孪生与楼宇可视化
- `case-04-*`：园区应急指挥

## 页面引用方式

- 截图访问路径：`/customer-cases/screenshots/文件名`
- 视频访问路径：`/customer-cases/videos/文件名`

当前页面入口：

- 中文：`docs/customer-cases/index.md`
- 英文：`docs/en/customer-cases/index.md`
